11.What are all the looping structures in JavaScript

A) there are three types of loop:-

(i)for loop
(ii)while loop
(iii)do-while looping

(i)for loop:-

for(var i=1;i<=10;i++){
    console.log(i);
}

(ii)while loop:-

var i=1;
while(i<=10){
    console.log(i);
    i++;
}

(iii) do-while loop

var i=1;
do{
    console.log(i);
    i++;
}while(i<=10);
