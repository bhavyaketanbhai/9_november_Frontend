4.What do you know about the Bootstrap Grid System?

A) ->The Bootstrap grid system is a responsive and mobile-first system that allows developers to create flexible and dynamic layouts for their web-pages.it is based on a 12-column grid,which can be customized to fit any design.the grid system works by dividing the page into rows and columns and then placing content within those column this allows for easy placement and alignment of the content and make it easy to create responsive design that look great on any device.
